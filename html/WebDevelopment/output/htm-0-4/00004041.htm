Step 2 - Target Connection Choose the target Oracle connection to use to migrate your database. \ Note: For single-schema migrations, your new objects will be created in this user's database. This user will need the following rights:\
  1. RESOURCE
  2. CREATE SESSION
  3. CREATE VIEW
For multi-schema migrations, this user will be used \ to create target databases, and it will need the following additional rights:\
CREATE USER, CREATE ROLE, ALTER ANY TRIGGER and RESOURCE all WITH ADMIN OPTION.
