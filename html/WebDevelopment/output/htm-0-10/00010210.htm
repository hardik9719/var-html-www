Installing Software Packages

Installing Software Packages

Linux operating system offers many ways to install software and libraries on the Operating System. In Ubuntu and Debian we have apt tool for installing, updating and removing the software packages.

Centos and Rehat Linux offers yum tool for installation of the software packages. The YUM stands for (Yellowdog Updater, Modified) and its command line tool for Package Management which was developed by RedHat.

Installing packages on Ubuntu

In Debian and Ubuntu we have APT (Advanced Packaging Tool) tool which is used most for management of packages on these operating system.
